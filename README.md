# Ejecutar en desarrollo

1. Clonar repo
2. Ejecutar

```
npm install
```

3. Tener nest cli instalado

```
npm i -g @nestjs/cli
```

4. Levantar BD

```
docker-compose-up -d
```

5. Stack usado

- MongoDB
- Nest
